A clean, minimal, and print-optimized A4 Landscape Habit Tracker built using pure HTML, CSS, and Vanilla JavaScript.

Designed for discipline. Built for clarity. Optimized for print.

📌 Overview

The Horizontal Ritual Tracker helps you track daily disciplines across an entire month in a beautifully structured horizontal grid format.

It automatically:

Adjusts days based on selected month

Highlights weekends

Saves your progress in browser

Generates a perfectly formatted A4 landscape printable sheet

No frameworks. No backend. No dependencies.

✨ Features

✅ Dynamic month & year selection

✅ Supports 28–31 days automatically

✅ Add unlimited disciplines

✅ Remove disciplines anytime

✅ Click-to-toggle completion circles

✅ Weekend highlighting

✅ Auto-save using browser localStorage

✅ A4 Landscape print optimization

✅ Clean editorial typography design

🖥️ Tech Stack

HTML5

CSS3 (Grid + Print Media Queries)

Vanilla JavaScript

Google Fonts (Playfair Display + Inter)

🖨️ Print Optimization

The project includes a dedicated print engine:

Forces A4 Landscape orientation

Removes UI controls while printing

Preserves circle fill colors

Maintains strong black borders
